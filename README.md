# Playwright Automation Framework

## Project Overview
This project is built using Playwright with TypeScript.

Framework includes:
- Page Object Model (POM)
- Custom Fixtures
- Hooks
- Parallel Execution
- Retry Mechanism
- Jenkins CI Integration
- HTML Reporting
- Screenshot on Failure
- Trace Viewer Debugging

---

## Test Scenarios
- Valid Login
- Invalid Login
- Empty Login
- Locked User Login
- Logout Flow

---

## Tech Stack
- Playwright
- TypeScript
- Node.js
- Jenkins
